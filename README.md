# Wireshark-Network-Analysis
Network Traffic Analysis using Wireshark


## Objective
The goal of this project is to analyze real network traffic and understand how common protocols such as DNS and HTTPS operate


## Tools used
* Wireshark


## Analysis performed
* Captured live network traffic 
* Applied filters to isolate DNS traffic
* Identified HTTPS encrypted sessions
* Examined packet details including IP addresses and ports


## Key Findings
* Observed DNS query and response process
* Identified client-server communication patterns
* Verified encrypted HTTPS traffic behavior
